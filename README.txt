Aplikacja umożliwiająca internetową sprzedaż produktów dla zwierząt domowych. 
Uruchomienie:
Wymagania:
Sql server 
Przed uruchomieniem projektu należy zmienić connectionstring (appsettings.json) oraz wykonać migracje do bazy danych. 
Dodatkowo należy do projektu doinstalować 4 pakiety NuGet:
Microsoft.AspNetCore.Identity.EntityFrameworkCore
Microsoft.EntityFrameworkCore
Microsoft.EntityFrameworkCore.SqlServer
Microsoft.EntityFrameworkCore.Tools

Po uruchomieniu aplikacji do dyspozycji zostało utworzonych dwóch użytkowników:
Administrator:
admin@storeapp.pl
Wsei!2022
